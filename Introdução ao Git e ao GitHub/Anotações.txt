# Como fazer o push de itens alterados do Local para o GitHub

git status (mostra o status dos itens atuais)
git add . (coloca os arquivos em estado de diponibilidade para upar, demonstrando a intenção de subir o arquivo)
git commit -m "Descrição da ação" (Realiza o commit do arquivo)
git push origin main (Realiza o push do commit)
